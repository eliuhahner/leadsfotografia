# Estratégia de Conversão - Rua do Outono

## Fluxo Completo

```
LEAD NO SITE
    ↓
Preencheu formulário → Sorteio → Vê resultado
    ↓                    ↓
    └──┬── Enviou WhatsApp? ──→ NÃO → Scripts Recuperação (24h/48h/72h)
           │
           └── SIM → Acolhimento → Qualificação → Upsell → Confirmação
```

## Scripts de WhatsApp
